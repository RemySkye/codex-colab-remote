# Troubleshooting

If local secret storage is unavailable on Linux, install and unlock a Secret Service-compatible desktop keyring. Plaintext fallback storage is intentionally unsupported.

Ask Codex to run `doctor`, `credential_status`, and `list_sessions` first. Do not post OAuth codes, tokens, session URLs, private keys, or personal logs.

See the [troubleshooting guide](https://github.com/RemySkye/codex-colab-remote/blob/main/docs/troubleshooting.md) for authentication, hardware capacity, local files, transfers, notifications, recovery, and SSH.
